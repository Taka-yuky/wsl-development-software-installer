# wsl-development-software-installer

1. GitHub CLIをインストール

```bash
sudo apt -y update
sudo apt -y install gh
```

2. GitHubへログイン
```bash
gh auth login
```

3. インストール用スクリプトを実行
```bash
cd wsl-development-software-installer
```

## 補足：gitの設定
```bash
  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"
```
